# dailymango-web

Dailymango 서비스 웹 페이지 — 랜딩, 앱 소개, 법적 문서

**도메인**: https://dailymango.rollingmango.co.kr

## 구조

```
/                          ← Dailymango 서비스 메인
/paperday/                 ← Paperday (페이퍼데이) 앱
  ├── index.html           ← 앱 소개
  ├── support.html         ← 고객 지원 + FAQ
  ├── privacy.html         ← 개인정보처리방침 (한)
  ├── privacy-en.html      ← 개인정보처리방침 (영)
  ├── terms.html           ← 이용약관 (한)
  └── terms-en.html        ← 이용약관 (영)
```
